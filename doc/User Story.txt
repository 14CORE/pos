User Story

Inventory System

- As a shop owner, I want to know the quantity of each item left in my shop so that I can plan to manage the stocks.
- As a shop owner, I want to manage items in the inventory.
- As a shop owner, I want to check the price of each item so that I can find and detect if the price is incorrect.
- As a shop keeper, I want to add a new item to the inventory.
- As a shop keeper, I can set the price of item.
- As a shop keeper, I want the to sort the inventory by name or price so that I can find items easily.

Sale System

- As a shop owner, I want sale system so that I can make a sale and serve a service conveniently.
- As a shop owner, I want the list of item that customer buy so that I can make sure that the sale processes correctly.
- As a shop keeper, I want the calculator that compute the total price of sale and change.

Basic user management system

- As a business owner, I want to add a new employee to the system.
- As a business owner, I can remove the user account of retired employee. 
- As a business owner, I want the working history of each employee so that I can know who is the best worker.
- As an employee, I can set the password for my account so that nobody else can use my account to do bad things.
- As a business owner, I want the login history so that I can trace an unusual activity.

Basic customer management system

- As a customer, I want to be a shop member so that I can get the privilege offers.
- As a customer, I want to edit information of my account.
- As a business owner, I want the customer analysis and segmentation so that I can develop a marketing plan.
