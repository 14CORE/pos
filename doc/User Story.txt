User Story

Inventory System

- As an owner, I want to know the quantity of each item left in my shop so that I can plan to manage the stocks.
- As an owner, I want to manage items in the inventory.
- As an owner, I want to check the price of each item so that I can find and detect if the price is incorrect.
- As an owner, I want to add a new item to the inventory.
- As an owner, I can set the price of item.
- As an owner, I want the to sort the inventory by name or price so that I can find items easily.
- As an owner, I want to change the currency so that I can accept vary type of currency.
- As an owner, I want the inventory change history so that I can trace and edit incorrect activity.


Sale System

- As an owner, I want sale system so that I can make a sale and serve a service conveniently.
- As an owner, I want the list of item that customer buy so that I can make sure that the sale processes correctly.
- As an owner, I want the calculator that compute the total price of sale and change.
- As an owner, I can override the sale price so that I can sale the item that discount for bargain situation. 

Basic user management system

- As an owner, I want to add a new employee to the system.
- As an owner, I can remove the user account of retired employee. 
- As an owner, I want the working history of each employee so that I can know who is the best worker.
- As an employee, I can set the password for my account so that nobody else can use my account to do bad things.
- As an owner, I want the login history so that I can trace an unusual activity.

Basic customer management system

- As an owner, I want the customer analysis and segmentation so that I can develop a marketing plan.
